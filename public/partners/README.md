# Partner logos

Drop each partner's logo here. Referenced from `src/data/partners.ts` via the
`logo` field (path relative to `/public`).

- **Sumware** → save as `sumware.png` (transparent background, ideally the
  horizontal wordmark, ~600×160px or larger). Referenced as `/partners/sumware.png`.

If a logo file is missing, `PartnerCard` automatically falls back to the
partner's name initials, so the directory never looks broken.
